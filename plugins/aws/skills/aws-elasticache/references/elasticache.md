# ElastiCache

Use this skill for:

- Redis or Memcached topology review
- replication and failover posture
- connection and timeout debugging
- auth, TLS, and subnet-group review
- eviction, TTL, and cache-key behavior

## Review Checklist

- cache engine and topology match workload shape
- subnet, SG, and route path allow reachability
- auth and TLS posture align with app configuration
- eviction policy and memory behavior fit the access pattern
