# API Gateway

Use this skill for:

- HTTP API and REST API route design
- Lambda or service integrations
- authorizers and token flow review
- CORS and browser-facing API issues
- stage variables, throttling, and deployment debugging

## Review Checklist

- route and method match
- integration target and payload format
- authn/authz behavior
- CORS headers and preflight
- custom domain, certificate, and DNS path
