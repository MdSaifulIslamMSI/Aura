# Event Driven

Use this skill for:

- SQS queue design and consumer tuning
- SNS fan-out patterns
- EventBridge routing and event contracts
- Step Functions orchestration
- retries, DLQs, ordering, and idempotency design

## Review Checklist

- producer contract and payload shape
- retry and backoff strategy
- DLQ or failure sink
- duplicate handling and idempotency
- observability around stuck or poisoned messages
