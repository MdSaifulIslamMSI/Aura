# S3

Use this skill for:

- bucket policy and access reviews
- presigned URL and upload flows
- static website and asset hosting decisions
- lifecycle, storage class, and retention planning
- event notifications into Lambda, SQS, or SNS

## Review Checklist

- Block Public Access state
- bucket policy and object ownership mode
- encryption at rest
- CORS for browser uploads
- lifecycle and versioning alignment with retention goals
