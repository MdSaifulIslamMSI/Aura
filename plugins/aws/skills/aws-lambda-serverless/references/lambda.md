# Lambda

Use this skill for:

- Function packaging and deployment shape
- Event source mapping and trigger debugging
- Environment variables and secret sourcing
- Timeout, memory, concurrency, and cold-start tradeoffs
- Lambda plus API Gateway or Function URL patterns

## Common Failure Buckets

- bad handler or runtime packaging
- missing trigger permission
- VPC networking or DNS issues
- env/config drift
- timeout and concurrency saturation
