# Frontend UI/UX Design Lab

`frontend-uiux-design-lab` is a reusable Codex plugin for public-facing frontend and UI/UX work.

It is designed to travel beyond a single app and help with:

- landing pages and marketing surfaces
- dashboards and information-heavy products
- forms, onboarding, and task flows
- ecommerce and marketplace interfaces
- design systems and token cleanup
- accessibility, motion, copy, and final polish

## Included Skills

- `art-direction-blueprints`
- `design-system-forge`
- `landing-page-storycraft`
- `dashboard-information-design`
- `forms-and-onboarding-flows`
- `responsive-layout-composition`
- `commerce-ux-polish`
- `ux-copy-ia-tuning`
- `motion-feedback-patterns`
- `accessibility-ui-review`
- `frontend-polish-pass`
- `ui-critique-and-redesign`

## Bundled Utilities

- `scripts/extract_design_tokens.py`
- `scripts/surface_brief.py`
- `assets/design-tokens-starter.css`
- `assets/ui-review-template.md`

## Reuse Notes

This plugin can live in a project-level `plugins/` directory or be copied into a home-level Codex plugin collection. The marketplace entry should point to `./plugins/frontend-uiux-design-lab`.
