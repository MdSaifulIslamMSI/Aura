# Dashboard Patterns

## Common Layers

- summary metrics
- trend visualizations
- filters and date range
- drill-down detail
- recent activity or alerts

## Layout Tips

- Put the most important trend near the top-left in left-to-right layouts.
- Keep comparison charts and tables close enough to support each other.
- Use one sticky filter bar instead of scattering controls.

## Visual Tips

- Use muted surfaces for scaffolding and stronger contrast for active data.
- Reserve saturated color for thresholds, anomalies, and selections.
- Let whitespace separate blocks before adding more borders.
