Curated access layer

Choose the host. Keep the same Aura reality.

This gateway is designed as the calmer, more deliberate front door above Aura's live storefront stack. Vercel and Netlify stay separate as public runtimes, but identity, cart, wishlist, checkout, profile, and order state still resolve into the same production experience.
