# Free Software Stack

Traffic fortress controls use free/open-source tools and free tiers:

- Express, Redis, MongoDB, Prometheus, Grafana, Alertmanager, OpenTelemetry.
- OWASP CRS, ModSecurity/Coraza-compatible examples, CrowdSec.
- k6 local/staging load drill scripts.
- CodeQL, Semgrep CE, OSV-Scanner, Trivy, Gitleaks, npm audit, Hadolint.
- Cloudflare Free or equivalent CDN/WAF free-tier controls where available.

Paid services are not required by this campaign.
